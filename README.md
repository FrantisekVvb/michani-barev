# Simulace míchání barev (RGB / CMY)

Jednoduchá webová aplikace pro **simulaci míchání barev** v modelech **RGB** (aditivní) a **CMY** (subtraktivní).

## Funkce

- **Procvičuj**: nastavuješ intenzity 3 kanálů a vidíš výslednou barvu.
- **Test**: náhodně zvolená cílová barva a úkolem je ji namíchat.
  - **Standard**: průběžně vidíš svoji namíchanou barvu.
  - **Expert**: svoji namíchanou barvu nevidíš až do vyhodnocení.
- **Vyhodnocení**: po kliknutí na **Vyhodnotit** se zobrazí shoda v % a správné nastavení.
